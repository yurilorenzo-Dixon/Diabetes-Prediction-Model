Load and preprocess the Pima Indians Diabetes dataset from OpenML Dataset ID 43483
Develop a Sequential neural network with Keras.
Train and evaluate the model on test data.
Save the final trained model in .keras format.

Features

Automatic dataset loading and target column detection.
Normalization of numeric data with StandardScaler.
Neural network built with TensorFlow/Keras:
Two hidden layers (12 and 8 neurons, ReLU activation).
One output layer (sigmoid activation).
Model evaluation using accuracy metrics.
Model saved in modern Keras format (.keras).
